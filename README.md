# 🔧 Pro Auto — Sistema de Gestão

Sistema de gerenciamento para oficina mecânica. Desenvolvido com React + Vite + Tailwind CSS.

## Módulos

- ⚡ **Dashboard** — visão geral, métricas e alertas
- 👥 **Clientes** — cadastro e busca de clientes e veículos
- 🔧 **Ordens de Serviço** — abertura, acompanhamento e conclusão de OS
- 📦 **Estoque** — controle de peças com alertas de estoque mínimo
- 💰 **Financeiro** — lançamentos de receitas e despesas

## Como rodar localmente

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Gerar build de produção
npm run build
```

## Deploy no Vercel

1. Suba este projeto no GitHub
2. Acesse [vercel.com](https://vercel.com) e importe o repositório
3. Clique em **Deploy** — pronto!
